# 🎬 C-Tube – Video Streaming Platform

## 📌 Overview

C-Tube is a full-stack video streaming web application that allows users to watch videos, interact through comments, and experience features like watch parties. The platform is designed to deliver a smooth and interactive video viewing experience similar to modern streaming platforms.

---

## 🚀 Features

* 📺 Video Streaming Interface
* 💬 Comments Section for user interaction
* 👥 Watch Party functionality (real-time interaction)
* 🎯 Responsive UI for better user experience
* 🔍 Video browsing and viewing pages

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Tailwind CSS / CSS
* React Router

### Backend

* Node.js
* Express.js

### Database

* MongoDB

### Version Control & Deployment

* Git & GitHub
* Vercel (Frontend Deployment)
